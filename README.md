# FlickrWall

Will fetch your groups photos and set as a background on your mac (sorry linux guys, maybe in the future)

## Installation

Add this line to your application's Gemfile:

    gem 'FlickrWall'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install FlickrWall
